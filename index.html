jQuery resizeEnd

A resizeEnd event for jQuery.

jQuery resizeEnd defines a special event that is fired when the JavaScript "resize" event has finished.

It also defines an alias function named after the event name which binds an event handler to the "resizeEnd" event, or trigger that event on an element.

Demo

Try to resize this window, the background color of the header will change at resizeEnd.

Usage

Basic usage

$(window).on("resizeEnd", function (event) {
  // go nuts
});

or use its alias function:

$(window).resizeEnd(function (event) {
  // go nuts
});
